⚙️ One-Time Google Cloud Setup

Go to console.cloud.google.com → create a new project (name it anything, e.g. outreach-tool).

APIs & Services → Library → search Gmail API → Enable it.

APIs & Services → OAuth consent screen → choose External → fill in an app name + your email as support and developer contact → Save and Continue through all steps.

APIs & Services → Credentials → Create Credentials → OAuth client ID → Application type: Web application.

Under Authorized JavaScript origins add these two URIs (click + ADD URI for each):
https://claude.ai
https://www.claude.ai

Leave Authorized redirect URIs empty. Click Create and copy the Client ID.

Paste the Client ID below and click Connect Gmail. A Google sign-in popup will appear — approve it. Done.
